# spring-jsp-crud
try to do crud with spring mvc and mybatis

	   sbt clean spring-boot:run


Then you can access them with urls:

* http://localhost:8080/register.jsp   
* http://localhost:8080/login.jsp
* http://localhost:8080/experts
